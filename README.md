# ScalaSVM: A scala port of Support Vector Machines

This is a Scala port of <a href="http://www.csie.ntu.edu.tw/~cjlin/libsvm/">libsvm</a>.

## Requirements
- Run: Java >= 1.6
- Compile: Java >= 1.6, Scala >= 2.9.2, Gradle >= 1.8.6

Developing
